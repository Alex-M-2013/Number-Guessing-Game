# Python3 Number Guessing Game
A simple command-line Number Guessing Game made in [Python3](https://www.python.org/)

- Uses the __random__ module 
- Tracks user attempts
- Checks if the user entered an integer between 1 and 10
- Asks if user wants to play again at end the of the game

To try it yourself, simply go to the [releases](https://github.com/Alex-M-2013/Number-Guessing-Game/releases/latest) section and download and run the .exe file.
