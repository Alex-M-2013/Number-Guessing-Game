# C# Number Guessing Game

A simple number guessing game made with .NET 10
This is my first C# project so the code is probably very inefficient and buggy 😅

- Tracks user attempts
- Checks if the user entered an integer between 1 and 10
- Asks if user wants to play again at end the of the game

To try it yourself, simply go to the releases tab and run the .exe file.
